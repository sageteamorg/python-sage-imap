Sync Tutorial
=============

Work through these lessons **in order**. Each lesson builds on the previous one. Code uses :class:`~sage_imap.session.IMAPSession` — the recommended synchronous API.

.. tip::

   New to Python? You only need basic ``with`` blocks, ``for`` loops, and ``try/except``. No Flask, Django, or databases.
